# Mallett Made Radiant Monitor

Live energy monitoring dashboard for radiant barrier installations.

## Stack
- **Frontend**: Next.js + React + Recharts
- **Backend**: Supabase (Postgres + REST API)
- **Hardware**: ESP32 + DS18B20 temperature sensors
- **Hosting**: Vercel

## Deploy
1. Push to GitHub
2. Connect repo to Vercel
3. Deploy (zero config needed)

## Custom Domain
Point `monitor.mallettmade.co` to Vercel in your DNS settings.
